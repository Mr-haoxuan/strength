# 基础库分离

1. html-webpack-externals-plugin
   可以使用 html-webpack-externals-plugin 将 react react-dom 等基础包通过 cdn 引入。不打包到 bundle 中。

```javascript
module.exports = {
  plugins: [
    new HtmlWebpackExternalsPlugin({
      module: 'react',
      entry: '//xxx',
      global: 'React',
    }),
  ],
}
```

2. webpack4 内置 SplitChunksPlugin 进行公共脚本分离（替代 CommonChunkPlugin）

```javascript
module.exports = {
  optimization: {
    splitChunks: {
      // async：异步引入的库进行分离（默认）
      // initial 同步引入的库进行分离
      // all 所有引入的库进行分离 （推荐）
      chunks: 'async',
      // 抽离公共包最小的大小 单位字节
      minSize: 30000,
      maxSize: 0,
      // 方法被引用 X 次后会被抽离
      minChunks: 1,
      maxAsyncRequests: 5,
      // 浏览器请求异步资源的次数
      maxInitialRequests: 3,
      automaticNameDelimiter: '~',
      name: true,
      checkGroups: {
        // 分离什么库
        test: /(react|react-dom)/,
        // 公共包名称
        name: 'vendors',
        chunks: 'all',
        minChunks: 2,
      },
    },
  },
}
```
